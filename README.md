# Voter_eligibility_test
# c programming
## 📌 Overview

This project is a simple C program that determines whether a person is eligible to vote based on their age. It is designed as a beginner-friendly program to demonstrate the use of basic programming concepts such as input/output operations, conditional statements, and user interaction.

Voting is a fundamental right in many democratic countries, and eligibility is generally determined by age. This program helps automate that check by asking the user to input their age and then displaying whether they are eligible to vote.

## 🎯 Objective

The main objective of this project is:

* To understand how conditional statements (`if-else`) work in C
* To practice taking user input using `scanf()`
* To display output using `printf()`
* To build logical thinking for simple real-world problems

## ⚙️ How It Works

1. The program prompts the user to enter their age.
2. The input age is stored in a variable.
3. The program checks whether the age is greater than or equal to 18.
4. If the condition is true, it prints that the user is eligible to vote.
5. Otherwise, it displays that the user is not eligible.

## 🧠 Concepts Used

* Data types (`int`)
* Input/Output functions (`printf`, `scanf`)
* Conditional statements (`if-else`)
* Basic program structure in C

## ▶️ Sample Output

Enter your age: 20
You are eligible to vote.

Enter your age: 16
You are not eligible to vote.

## 🚀 How to Run

1. Save the file as `voter.c`
2. Compile using a C compiler:

   ```
   gcc voter.c -o voter
   ```
3. Run the program:

   ```
   ./voter
   ```

This project is a great starting point for beginners learning C programming. It demonstrates how simple logic can be applied to solve real-life problems efficiently.
